Co trafia do zbioru YES?

1. Czy dane odnoszą się do jednostek przestrzennych?
np. państwa, województwa, powiaty, siatka, izolinie
- Jeśli tak → idziemy dalej.
- Jeśli nie → mapa raczej nie jest statystyczna.

2. Czy dane są policzalne / pochodzą ze statystyki?
np. liczba ludności, PKB, liczba głosów, powierzchnia upraw
- Jeśli TAK → prawdopodobnie statystyczna.
- Jeśli NIE → np. typ gleby, rodzaj skały → nie statystyczna.

3. Czy przedstawiają zjawiska masowe lub populacyjne?
masowe = obejmujące wiele jednostek / liczbę osób lub obiektów.
- Jeśli tak → statystyczna.

4. Czy dane są ilościowe lub jakościowe, ale wynikają ze statystyki?
- Ilościowe → liczby, wskaźniki, gęstość
- Jakościowe statystyczne → np. dominująca partia, struktura wykształcenia
- Jeśli dane jakościowe nie wynikają ze statystyki, np. typ gleby → nie statystyczna

5. Czy celem mapy jest pokazanie wzorców liczbowych lub struktury w przestrzeni?
- Jeśli tak → statystyczna.
- Jeśli mapa tylko pokazuje lokalizację lub przebieg (drogi, rzeki) → nie statystyczna.


- kartogramy ilościowe
- kartogrmay porządkowe
- kartogramy jakościowe, ale statystyczne (np. dominująca partia, struktura wykształcenia)
- mapy z izoliniami, jeśli przedstawiają dane statystyczne (np. gęstość zaludnienia, temperatura)
- mapy odniesione do jednostek geometrycznych, ale pokazujące dane statystyczne (np. liczba ludności, PKB, liczba głosów, powierzchnia upraw)
- kartodiagramy
- kartogramy anamorficzne
- minimum 50% obrazka zajmuje mapa statystyczna
- jeśli mapa ma wiecej niż jedną metodę (ale conajmniej jedna spełnia nasze założenia) → YES
- jesli mapa jest konturowa lub z jakimś wypełnieniem ale nie wiemy co przedstawia -> NO
- jeśli jestesmy mocno niepewni -> NO
