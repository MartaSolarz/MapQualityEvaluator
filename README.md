# MapQualityEvaluator

Model GenAI do ewaluacji jakości map statystycznych

## **🎯 Cel badania**
Opracowanie i walidacja metodyki oceny jakości map statystycznych z wykorzystaniem GenAI (Vision Language Models) wraz z utworzeniem publicznie dostępnego benchmark datasetu dla tego zadania.


## **✅  Planowane efekty (Deliverables):

### 1. **Framework - zasady redakcji map statystycznych**
- Przygotowany na podstawie literatury
- Zwalidowany przez ekspertów (5-10 kartografów z całego świata?)
- Zasady będą miały postać hierarchicznego frameworka z przypisanymi wagami dla każdej kategorii (patrz Podejscie_teoretyczne.md)
- **Wartość z punktu widzenia kartograficznego**

### 2. **StatMapQA Dataset**
- Kilka tysięcy map statystycznych z pełnymi adnotacjami - pochodzą z datasetu MapPool (wybrane z wielu milionów, te które spełniają nasze kryetria)
- Zbalansowany pod względem: typu mapy, jakości, reprezentacji błędów
- Publicznie dostępny z DOI (Zenodo/HuggingFace)
- Pełna dokumentacja
- **Wartość z punktu widzenia informatycznego

### 3. **Model oceniający jakość map**
- **Wartość z punktu widzenia kartograficznego/informatycznego**
- Oparty na GenAI (Vision Language Models)

### 4. **Publikacja
- Metodyka + wyniki eksperymentów + analiza
- Limitations & future work
